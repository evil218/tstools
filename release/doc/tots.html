NAME

tots - manual page for tots of tstools 1.0.0  

SYNOPSIS

tots [OPTION] file [OPTION]  

DESCRIPTION

'tots' read from stdin, translate 'XY ' to 0xXY, send to file.  

OPTIONS

-h, --help
print this information, then exit
-v, --version
print my version, then exit
 

EXAMPLES

tots xxx.ts

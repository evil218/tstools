NAME

tobin - manual page for tobin 1.0.0  

SYNOPSIS

tobin [OPTION] file [OPTION]  

DESCRIPTION

'tobin' read from stdin, translate 'XY ' to 0xXY, send to file.  

OPTIONS

-h, --help
print this information, then exit
-v, --version
print my version, then exit
 

EXAMPLES

tobin xxx.ts

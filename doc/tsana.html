NAME

tsana - manual page for tsana of tstools v1.1.384 (384m 63a7d3a8)  

SYNOPSIS

tsana [OPTION]...  

DESCRIPTION

'tsana' get TS packet from stdin, analyse, then send the result to stdout.  

OPTIONS

-lst
show PID list information, default option
-psi
show PSI tree information
-expsi
export PSI information into psi.xml
-impsi
import PSI information from psi.xml before analyse
-dump
dump cared packet
-mem
memory pool status show level[none|total|detail], default: none
-time
"*time, YYYY-mm-dd HH:MM:SS, second, usecond, delta_time(ms), "
-addr
"*addr, address(hex), address(dec), PID, "
-cts
"*cts, CTS, BASE, "
-stc
"*stc, STC, BASE, "
-pcr
"*pcr, PCR, BASE, EXT, dSTC(ms), dPCR(ms), PCR-STC(ns), "
-pts
"*pts, PTS, dPTS(ms), PTS-PCR(ms), DTS, dDTS(ms), DTS-PCR(ms), "
-tsh
"*tsh, 47, xx, xx, xx, "
-ts
"*ts, 47, ..., xx, "
-mts
"*mts, 3F4BD, "
-af
"*af, xx, ..., xx, "
-pesh
"*pesh, xx, ..., xx, "
-pes
"*pes, xx, ..., xx, "
-es
"*es, xx, ..., xx, "
-sec
"*sec, interval(ms), head, body, "
-si
"*si, interval(ms), head, information of body, "
-rate
"*rate, interval(ms), PID, rate, [es_rate, ]..., PID, rate, [es_rate, ]"
-rats
"*rats, interval(ms), SYS, rate, PSI-SI, rate, 0x1FFF, rate, "
-ratp
"*ratp, interval(ms), PSI-SI, rate, PID, rate, ..., PID, rate, "
-err
"*err, TR-101-290, datail, "
-c -color
enable colour effect to help read, default: mono
-start <x>
analyse from packet(x), default: 0(first packet)
-count <n>
analyse n-packet then stop, default: 0(no stop)
-pid <pid>
set cared PID[0x0000,0x2000], default: 0x2000(any PID)
-table <id>
set cared table[0x00,0xFF], default: 0xFF(any table)
-prog <prog>
set cared prog[0x0000,0xFFFF], default: 0x0000(any program)
-type <type>
set cared PID type[any|vid|aud|emm|ecm], default: any
-iv <iv>
set cared interval(1-70000)ms, default: 1000(1000 ms)
-mp <mp>
set memory pool size order(16-32), default: 20, means 2^20 bytes
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

"catts xxx.ts | tsana -c -time -addr -pcr -pts" -- report all PCR/PTS/DTS information

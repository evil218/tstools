NAME

catts - manual page for catts of tstools v1.0.0 (354 81b96b4c)  

SYNOPSIS

catts [OPTION] file [OPTION]  

DESCRIPTION

'catts' read binary file, translate 0xXY to 'XY ' format, then send to stdout.  

OPTIONS

-w, --width <n>
n-byte per line for FILE_BIN, default: 16
-s, --start <a>
cat from, default: 0(from first byte)
-p, --stop <b>
cat to, default: 0(to last byte)
-l <level>
set report level(dbg|inf|wrn|err), default: wrn
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

catts xxx.ts

NAME

toip - manual page for toip of tstools v1.1.387 (387m 0fe0e32d)  

SYNOPSIS

toip [OPTION] udp://@xxx.xxx.xxx.xxx:xxxx [OPTION]  

DESCRIPTION

'toip' read from stdin, convert to UDP, send to IP according to ATS.  

OPTIONS

-h, --help
print this information only
-v, --version
print my version only
 

EXAMPLES

catts *.mts | toip udp://@:1234
catts *.mts | toip udp://@224.165.54.210:1234
catts *.ts | tsana -ts -ats | toip udp://@:1234
